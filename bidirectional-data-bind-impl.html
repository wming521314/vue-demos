notice:

1. In function defineProperty , set and get will be triggered automatically when data obj changed , without binding event listener

2. If the property of obj is added without defineProperty , the set and get will not be triggered . In other words , the set and get functions of every property must be added respectively.

3. the expression : obj.user_name = "robin" , will not be added to obj , you can console.log(obj) to prove it , but the obj has set and get functions about user_name (Obviously , these functions come from defineProperty).
